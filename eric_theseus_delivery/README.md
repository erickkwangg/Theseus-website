# Theseus

## Technology used

- Next.js
- TailwindCSS
- ShadCN
- Typescript

## Local installation guideline

First clone the project and install dependencies

```bash
npm run install
# or
yarn
# or
pnpm install
# or
bun install
```

Then run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
